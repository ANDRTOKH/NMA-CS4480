# Network Managment Automation (PA3)

<b> Run the setup script: </b>
- chmod +x setup
- ./setup

<b> Use the orchestrator: </b>
- To switch to the North path (via R2):
    <t> ospfroute north </t> 
- To switch to the South path (via R4):
  <t> ospfroute south </t> 
- To Show help message
  <t> ospfroute -h </t>
